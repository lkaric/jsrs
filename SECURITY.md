# Security Policy

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Email security issues to: security@jsrs.dev

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations

You will receive a response within 48 hours. We aim to release a fix within 14 days for critical issues.

## Scope

- ZKP anonymity guarantees (highest priority)
- Authentication and session management
- SQL injection / XSS / CSRF
- Data exposure across the auth/anon boundary

## Out of Scope

- Denial of service attacks
- Social engineering
- Physical security
